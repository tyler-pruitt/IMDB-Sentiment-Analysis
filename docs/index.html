Chinese Character Recognition

Hello! This is where I hope to host my machine learning model in the future. Project hosted at: https://github.com/tyler-pruitt/Chinese-Character-Recognition. The idea is to create a space (i.e. a 64x64 pixel box) in which anyone can write one of the Chinese characters that the model has been trained on and it will make a prediction! I also plan to showcase the probabilities with a numeric and graphical output of the model's prediction to show some cool statistics! Please keep coming back to see updates and progress!








To use:
1) Write the character in the black box below. Available characters are 零，一，二，三，四，五，六， 七，八，九，十，百，千，万，and 亿. Attempt to draw the character as close to the sample data shown above as the model is sensitive to the way in which the character is written.
2) Press the 'Predict' button to have the model make a prediction on your writing.
3) Scroll down to see predicted character along with probability and the bar chart showing of the probabilities for all of the characters.
